# LearnIT - Online Courses Landing Page

LearnIT is a modern, responsive, and visually appealing landing page designed for an online courses platform. It is built using clean HTML and CSS, featuring a unique "Warm Paper & Sage" color theme to create a calm, distraction-free studying environment.

## 🌟 Features

- **Responsive Design**: Fully responsive layout that looks great on mobile, tablet, and desktop devices.
- **Calm UI Aesthetics**: Features a relaxing oatmeal and sage green color palette, soft inset shadows, and smooth hover animations for a premium, focused feel.
- **Hero Section**: An engaging intro section with clear Call-to-Action (CTA) buttons and a dynamic floating mock course UI.
- **Why Choose Us (Bento Layout)**: A modern grid-based "Bento Box" features section highlighting the benefits of the platform. The cards use custom inset styling inspired by Uiverse.
- **Interactive Courses Accordion**: A highly interactive expanding flex-accordion layout displaying popular courses. When hovered, the course cards dynamically expand to reveal categories, ratings, and pricing (localized in INR - ₹).
- **Call-to-Action**: A distinct floating bottom banner section to encourage user sign-ups.

## 🛠️ Technologies Used

- **HTML5**: Semantic and structured markup.
- **CSS3**: Custom properties (variables), Flexbox, CSS Grid, media queries, and animations. No external UI frameworks (like Bootstrap or Tailwind) were used to maintain maximum control over the design.
- **Google Fonts**: Uses the 'Outfit' font family for clean and modern typography.

## 🚀 How to Run Locally

Since this is a static webpage without any complex build steps or dependencies, running it is incredibly simple:

1. Clone or download this repository/folder to your local machine.
2. Open the folder containing the project files.
3. Double-click on `index.html` to open it in your default web browser.

Alternatively, if you use VS Code, you can install the **Live Server** extension and click "Go Live" to run the project with automatic reloading.

## 🎨 Theme & Colors

The landing page uses a carefully selected "Oatmeal & Sage" theme to promote a sense of calm focus, reducing eye strain and creating an optimal "studying" atmosphere.

- **Background**: Soft Warm Paper / Oatmeal (`#fdfbf7`)
- **Primary Accent**: Muted Sage Green (`#7ba098`)
- **Text Color**: Calm Dark Slate / Charcoal (`#2c3e50`)
- **Card Backgrounds**: Light Gray (`rgb(236, 236, 236)`) with complex inset shadows.
