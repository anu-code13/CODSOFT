
# Personal Portfolio - Anvesha Munankar

This is a clean, modern, and responsive personal portfolio web application built with HTML5 and CSS3. It serves as a personal space on the internet to showcase my skills and projects as a BSc IT student exploring web development.

## Features

- **Responsive Design**: Looks great on both desktop and mobile devices.
- **Modern Aesthetics**: Utilizes dynamic hover states, smooth transitions, and a dark mode theme with vibrant accents.
- **Custom Fonts**: Incorporates Google Fonts (`Outfit` and `Inter`) for clean and professional typography.
- **Sections**:
  - **Header**: Introduction and navigation.
  - **About**: Profile picture (`profile.jpeg`), a brief bio, and a downloadable PDF resume.
  - **Skills**: A grid layout displaying expertise areas (HTML, CSS, Python, Linux).
  - **Projects**: Cards showcasing my projects (Portfolio and Course Website) along with the specific tech stacks used.
  - **Contact**: Quick contact information for networking.

## Project Structure

```
portfolio/
├── portfolio.html # The main HTML document containing the structure and content
├── style.css      # The stylesheet containing all design tokens and responsive rules
├── resume.html    # A printable, professionally styled HTML resume
├── resume.pdf     # The generated one-page PDF version of the resume
├── profile.jpeg   # Personal profile image used in the 'About' section
└── README.md      # This file
```

## How to Run

1. Simply open the `portfolio.html` file in your preferred web browser. No local server or build tools are required!
2. To edit or update the resume, edit `resume.html`, open it in a browser, and use "Print to PDF" to replace `resume.pdf`.
