# Premium Light Blue Calculator

A modern, visually striking web calculator built as part of an internship project. It features a beautifully clean, crisp light blue and white aesthetic, offering a fresh, soft, and modern UI.

## 🌟 Features
- **Unique Design**: A premium, soft light blue interface featuring crisp white circular keys and a distinct rounded white display area for clear readability.
- **Dynamic CSS Grid Layout**: Clean and responsive alignment for the keypad buttons (e.g., `AC` and `0` buttons spanning multiple columns).
- **Standard Operations**: Addition (+), Subtraction (−), Multiplication (×), and Division (÷). Includes decimal support and zero division error handling.
- **Interactive Feedback**: Satisfying, snappy hover and active states using modern CSS box-shadows to simulate tactile button presses.

## 🛠️ Technology Stack
- **HTML5**: Semantic structure to manage the calculator display and keys securely.
- **CSS3**: Implementation of CSS Grid, Flexbox, custom `@keyframes` for subtle ambient background pulses, and modern soft-UI shadows.
- **Vanilla JavaScript**: Object-oriented implementation focused on state management without external libraries.

## 💻 Technical Requirements Fulfilled
- **CSS Grid**: Implemented in `.keypad` for aligning keys beautifully.
- **Event Listeners**: Used on all keys to detect user interaction.
- **Loops**: Used a `for` loop to efficiently traverse the DOM NodeList and apply event listeners.
- **If-Else Statements**: Extensively mapped for operator mapping, input routing, computation logic, and input validations.
- **Operators**: Performs math calculation directly via JavaScript's built-in mathematical operators `+`, `-`, `*`, `/`.

## 🚀 How to Run
1. Navigate to this folder (`calculator`).
2. Double click on `calculator.html` to open it in any modern web browser (Chrome, Firefox, Safari, Edge).
3. Experience the fluid layout and functionality!
